# New feature: Profile Pages

Should contain profile information about both users and artists, including:
- Profile picture
- Username
- Bio/description

For artists, the profile page should also include a gallery of their artworks and reference art pieces with pagination. Each artwork should be displayed as a thumbnail, and clicking on it should take the user to the artwork detail page.

Tabbed view of following sections:
- Artworks Tab: A gallery of the artist's artworks with pagination.
- Reference Arts Tab: Completed reference art pieces that the artist has created, with pagination. 

Note: Leave Reference Art Tab content empty for now, we will make it functional in a later feature.

Profile pages should be accessible via the URL structure: /users/{username} for regular users and /artists/{username} for artists. The profile page should be designed to be visually appealing and user-friendly, allowing visitors to easily navigate through the artist's portfolio and learn more about them.

Profile pages should be linked from various parts of the website, such as artwork detail pages, comments, and any other places where the artist's username is displayed. This will allow users to easily access the artist's profile and explore their work.
